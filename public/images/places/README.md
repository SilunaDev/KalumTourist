# Places Images Folder

## Instructions
Add images of places you've visited in Sri Lanka. Name them sequentially:

- **place1.jpg** - First destination (e.g., Sigiriya)
- **place2.jpg** - Second destination (e.g., Galle Fort)
- **place3.jpg** - Third destination (e.g., Ella)
- **place4.jpg** - Fourth destination (e.g., Yala)
- **place5.jpg** - Fifth destination (e.g., Kandy)
- **place6.jpg** - Sixth destination (e.g., Mirissa)

### Image Specifications:
- **Format**: JPG, PNG, or WEBP
- **Recommended Size**: 800x800px (square) or 1200x800px (landscape)
- **File Size**: Keep under 500KB each for optimal performance
- **Quality**: High resolution for clarity

### Tips:
- Use beautiful landscape or landmark photos
- Good lighting and clear composition
- Showcase the best of each location
- You can add more places by adding more images and updating the component

### Current Structure:
```
public/
└── images/
    └── places/
        ├── place1.jpg (Destination 1)
        ├── place2.jpg (Destination 2)
        ├── place3.jpg (Destination 3)
        ├── place4.jpg (Destination 4)
        ├── place5.jpg (Destination 5)
        └── place6.jpg (Destination 6)
```

**Note**: You can edit the place names and descriptions in the Places.tsx component file. Locations are positioned randomly around the map as requested.
