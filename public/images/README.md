# Images Folder

## Logo
Place your logo image in the `logo` folder:
- Recommended format: PNG with transparent background
- Recommended size: 512x512px or larger
- Name it: `logo.png`

## Carousel Images
Place your carousel images in the `carousel` folder:
- Add 4 images named: `slide1.jpg`, `slide2.jpg`, `slide3.jpg`, `slide4.jpg`
- Recommended format: JPG or PNG
- Recommended size: 1920x1080px (Full HD) for best quality
- Keep file sizes optimized (under 500KB each for better performance)
